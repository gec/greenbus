Distribution Directory

In the distribution tarball are the following directories:

bin/

Contains the greenbus script used to run services and utilities.

etc/

Contains configuration for processes.

install/

Contains a script, install_service_alias.sh that creates a Linux init.d service for a GreenBus service.

lib/

Contains binary libraries and the configuration necessary to start Java processes.

samples/

Model XML samples.

var/

Contains log files under var/log/[process name]/.

Running processes

To run service (background) processes:

bin/greenbus services start

To run utility processes:

bin/greenbus import config.xml

To see help for utility processes:

bin/greenbus export --help
