GreenBus is an open source telemetry control platform designed to enable interoperability and coordination between remote devices, automation applications, and human operators.

Features

  • Service APIs provide a common interface regardless of data sources or storage technology.
  • Scale by distributing load across stateless services, naturally partitioning data sources, and utilizing highly scalable time-series storage.
  • A fine-grained user authorization system on top of transport encryption and authentication.
  • Adapters available for the DNP3 and Modbus protocols.
